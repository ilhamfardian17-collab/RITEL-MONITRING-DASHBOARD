📊 Hubungkan Google Sheets

Paste link Google Sheets kamu. Semua user yang buka dashboard ini akan melihat data yang sama secara real-time.

Cara mendapatkan link:
1. Buka Google Sheets → File → Share → Publish to web
2. Pilih Sheet yang berisi data → Format: Comma-separated values (.csv)
3. Klik Publish → Copy link → Paste di atas
💡 Atau share biasa: klik Share → Anyone with link → Copy link
